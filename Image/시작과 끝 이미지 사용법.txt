이미지 저장
생성된 이미지를 다운로드하고 .png 형식으로 저장하세요. 이 이미지는 유니티에서 UI 요소로 사용할 수 있습니다.

유니티에 이미지 추가

유니티 프로젝트 폴더에서 Assets 폴더를 엽니다.
저장한 .png 파일을 Assets 폴더 안에 드래그 앤 드롭하여 가져옵니다.
Sprite로 설정
유니티에서 이미지를 선택한 후 Inspector 창에서:

Texture Type를 Sprite (2D and UI)로 변경.
변경 후 Apply 버튼을 클릭하여 저장.
UI 배치

Canvas를 생성하거나 기존 Canvas를 사용합니다.
Image UI 요소를 추가한 뒤, 가져온 이미지를 Source Image로 설정합니다.
버튼 추가

유니티에서 텍스트나 버튼을 추가하려면:
Canvas 하위에 Button을 생성.
버튼의 Image 컴포넌트를 비활성화하고, 원하는 텍스트 스타일을 적용합니다.
스크립트를 통해 버튼의 클릭 이벤트를 연결하세요.
